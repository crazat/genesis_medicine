# 한국 의료 AI 기업 협업 분석 (2026-04 기준)

## 1. 표준 사례 — Lunit (검증된 한국 의료 AI 사업화 모델)

### 핵심 지표 (2025)
| 지표 | 값 |
|---|---|
| 매출 (2021 → 2025) | ₩6.6B → **₩83.1B** (12×) |
| 진출 국가 | **50+ 국가** |
| FDA 진입 | Lunit INSIGHT Risk (breast cancer prediction) submitted |
| 시가총액 (2024 IPO 후) | ~$1.5B+ |
| 핵심 제품 | Lunit INSIGHT (cancer imaging — mammography, CXR, pathology) |

### 사업화 표준 process (Lunit 사례 분석)
1. **2013-2018**: 임상 evidence 축적 (FDA 510(k), CE-mark 등)
2. **2019-2021**: 한국 GMP 병원 도입 (서울대, 연세대, 삼성)
3. **2022-2023**: 글로벌 distribution partnership (GE, Philips, Guardant Health $26M)
4. **2024**: IPO (KOSDAQ 328130)
5. **2025**: 미국 시장 진입, FDA premarket notification

→ **Recover + Genesis_Medicine v3 적용**: 12-18개월 단축 가능 (Lunit 모델 학습)

## 4. 협업 모델 (3가지 시나리오)

### Scenario A: 독립 사업화 (현재 default)
- Recover 한의원 + Genesis_Medicine v3 단독 운영
- NIPA 2025 응모 + 자체 IND
- 12-18개월 1차 lead 출시
- **장점**: IP 100% 보유, 의사결정 속도
- **단점**: 자본 한계, 마케팅 채널 부족

### Scenario B: Lunit 협업 (imaging 통합)
- Recover의 AI 안면분석 ↔ Lunit imaging 기술 협업
- Genesis_Medicine v3 분자 처방 ↔ Lunit 글로벌 채널
- **장점**: 글로벌 50개국 즉시 진입 가능
- **단점**: 협상 → 의사결정 12개월+

### Scenario C: 한방 의료 AI 컨소시엄 (KIOM 주도)
- 한국한의학연구원 + 여러 한의원 (Recover 포함) + Genesis_Medicine v3
- 정부 사업 (NIPA + 보건복지부 + 식약처)
- **장점**: 정책 지원 + 공동 IRB
- **단점**: 의사결정 늦고 IP 분산

**권장**: **A → B 단계** (독립 1년 → 글로벌 협업)
