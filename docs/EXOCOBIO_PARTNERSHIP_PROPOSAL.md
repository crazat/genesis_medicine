# ExoCoBio × Recover 협업 제안서

## 1. Executive Summary

**제안**: ExoCoBio의 **adipose stem cell exosomes (ASC-Exos)** + Genesis_Medicine v3의 **EMB-3 small molecule** + Recover 한의원 **자운고 한방 외용제**의 **triple combination**으로 위축성 + 비후성 흉터 동시 적응증 공동 개발.

**가치**:
- ExoCoBio: 글로벌 stem cell exosome 1위, 8 publications + 48 patents (8 US), **세계 최초 split-face RCT** (acne scar)
- EMB-3: ABFE -32.9 kcal/mol, hERG -61%, MD 0.79 Å, network 6/7 anti-fibrotic 타겟
- 자운고: 한국 한방 흉터 외용 표준
- **Triple synergy**: scarless cutaneous repair (Exos) + anti-fibrotic (EMB-3) + 한방 baseline (자운고)

### ASCE+ Series
- ASC-Exos topical cream
- 한국 + 글로벌 미용 시장 진출

   - miR cargo: miR-181a/SIRT1, miR-192-5p/IL-17RA/Smad, miR-425-5p/142-3p (TGF-β1 차단)

## 4. 협업 모델

### Model A: 정보 교환 + Co-formulation 공동 개발 (3-6개월)
- ASC-Exos topical cream + EMB-3 + 자운고 안정성 평가
- 한국 CRO에 in vitro IC50 (TGFB1 reporter) ₩30M
- 우리 EMB-3 ABFE 데이터 + ExoCoBio miR cargo 매핑
- **Cost**: ₩50-80M (split 50:50)
- **Timeline**: 6개월

### Model B: Combination Phase 1 임상 (12-18개월)
- 흉터 환자 30-50명 split-face RCT
   - Group 1: ExoCoBio 단독 (control)
   - Group 2: ExoCoBio + EMB-3 외용
   - Group 3: ExoCoBio + EMB-3 + 자운고 (triple)
- Recover 한의원 환자 모집 (강남 base)
- Endpoint: PSCR 12-week 개선
- **Cost**: ₩200-300M (정부 지원 + 자부담)
- **Outcome**: Group 3 효능 우월 시 → Phase 2 확장

### Model C: K-MEDI Hub 컨소시엄
- ExoCoBio + Recover + KIOM + KOTRA
- 한국 한방 + 줄기세포 + AI 통합 글로벌 진출
- **Timeline**: 24-36개월
- **Outcome**: K-MEDI 대표 사례 + 글로벌 license
