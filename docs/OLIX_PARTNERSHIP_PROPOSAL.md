# OliX × Recover 협업 제안서

## 1. Executive Summary

**제안**: OliX Pharmaceuticals의 **OLX101 (CTGF asiRNA)** + Recover/Genesis_Medicine v3의 **EMB-3 (1,4-benzoquinone small molecule)** 통합 처방을 hypertrophic scar 적응증에 공동 개발.

**가치**:
- OLX101: CTGF mRNA silencing — 10-30% scar volume 감소 (Phase 1 호주)
- EMB-3: TGFB1/MMP1/CTGF/Smad3/PDGFRB 5-tier mechanotransduction 차단
- **Combination**: mRNA silencing (OLX101) + protein binding (EMB-3) → **dual-axis fibrosis 차단**
- 시너지 가설: CI < 0.85 (EMB-3 alone IC50 + OLX101 alone IC50 → combo 5-10× 강함)

## 2. 우리 자산 (Recover + Genesis_Medicine)

### EMB-3 정량 데이터
- **ABFE ΔG_decoupling = −32.9 ± 0.30 kcal/mol** (RTX 5090 80 ns)
- **MD 0.79 Å mean RMSD** (MMP-1, EGCG 1.45 Å보다 안정)
- **Boltz-2 affinity**: TGFB1 0.749, MMP1 0.674, CTGF 0.678, SMAD3 0.649, PDGFRB 0.640
- **ADMET**: hERG 0.16 (Embelin 0.40 대비 -61%), Skin_Reaction 0.67 (-20%), logKp -1.86 (외용 sweet spot)
- **Cross-disease**: IPF 6/7 타겟, scleroderma 7/7

## 4. 협업 모델 (3가지)

### Model A: 비임상 데이터 공유 (Phase 1)
- 우리 EMB-3 in vitro IC50 + ABFE 데이터 → OliX
- OliX OLX101 wet lab 데이터 → 우리
- 각자 IP 보유, 정보 교환만
- **Cost**: ₩0 (information exchange)
- **Timeline**: 3개월 NDA 후

### Model B: Combination 임상 (Phase 1/2)
- OLX101 + EMB-3 외용 combination clinical trial
- Recover 한의원 환자 모집 (강남 base + 협력 한의원)
- 흉터 환자 30-50명 12-week trial
- **Cost**: ₩500M (split 50:50 또는 OliX 70% / Recover 30%)
- **Timeline**: 18-24개월 (IRB + 환자 모집)
- **Outcome**: PSCR 50% 개선 시 → Phase 2 확장

### Model C: 공동 신약 회사 설립
- "OliX-Recover Joint Venture" (전통한방 + RNAi 통합 외용제)
- 1차 lead: OLX101 + EMB-3 + 자운고 추출 hybrid
- 한국 임상 → 글로벌 진출
- **Cost**: ₩수십억 (시드 + Series A)
- **Timeline**: 36-48개월
- **Outcome**: KOSDAQ 상장 또는 글로벌 license
