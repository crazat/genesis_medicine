# Recover PBM LED 시술 프로토콜

## 1. 배경

**Photobiomodulation (PBM)** = 적색/근적외선 빛을 피부에 조사 → 미토콘드리아 cytochrome c oxidase 활성화 → ATP ↑ + ROS 신호 → 콜라겐 합성·항염·재생.

**한국 임상 입증** (J Korean Soc Laser Med Surg):
- CO2 laser drilling + 830 nm Omnilux PDT LED → keloid 효능
- AAD 2024 consensus → 흉터, 여드름, 광노화 임상 표준

**Stanford 2025 + Wiley 2025**:
- Red light (620-700 nm): 콜라겐 합성·anti-aging
- NIR (830 nm): 심부 침투 (4-5 mm), 흉터 재구성
- Blue (415 nm): C. acnes 살균 (여드름 직접)

## 2. Recover 시술 통합 프로토콜

### Phase 1: 시술 전 준비 (5분)
- 환자 사진 (smartphone app, T6-4)
- AI 안면 분석 → 처방 확정 (Genesis_Medicine v3)
- 환자 PSCR 베이스라인 기록

### Phase 2: 약침 시술 (15분)
- 새살침 → 흉터 부위 자극
- 약침 cocktail 주입:
  - 비후성: EMB-3 + Embelin + 황련해독탕 추출
  - 위축성: Asiaticoside + Pro-Hyp + 자운고 추출
  - 여드름: 황련해독탕 + EMB-3 (low dose)

### Phase 3: PBM LED 즉시 적용 (20분)
- **Wavelength 선택** (질환별):

| 질환 | 1차 wavelength | 2차 | Power | Duration |
|---|---|---|---|---|
| 비후성 흉터 | **830 nm NIR** (deep) | 660 nm (combo) | 50 mW/cm² | 20분 |
| 위축성 흉터 | **660 nm red** (collagen) | 830 nm | 50 mW/cm² | 20분 |
| 여드름 | **415 nm blue** (C. acnes) | 660 nm | 30 mW/cm² | 15분 |
| 광노화 | 660 nm + 830 nm dual | — | 50 mW/cm² | 20분 |
| 탈모 | 660 nm scalp | 830 nm | 50 mW/cm² | 25분 |

- **Photoporation 효과**: LED light가 외용제 침투 가속 (40-60% ↑)

### Phase 4: 자운고/EMB-3 외용제 적용 (5분)
- LED 직후 외용제 → 침투 극대화
- Recover Triple Cream (T7-4 처방):
  - Universal / Atrophic / Hypertrophic 3 라인

### Phase 5: 가정 follow-up
- Smartphone app 매일 사진
- LED home device (option) 660 nm 5분/일
- 4-12주 추적 → AI 자동 효능 평가
